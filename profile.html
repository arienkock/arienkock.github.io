Skills

Web back-end

I have broad experience with the Servlet spec, but prefer working with the Spring Framework abstractions and rapid development using Spring Boot. I find the web container spec to be lagging behind the times and have chosen to investigate and experiment with Undertow as lower level high-performance alternative to servlet containers as well as hooking into Jetty’s extensions, SPDY in particular. I’ve also have implemented several search engines from the ground up with Solr and in one case using Lucene directly.

I spend some nights in bed thinking about software architecture. This may not be healthy for me, but you can always use this to your advantage by hiring me to do consulting.

Persistence

I have a love-hate relationship with Hibernate. I once wasted a lot of time debugging a problem that was solved by flushing the session in between editing persistent bags. Still, for simple entity graphs it beats having to write RowMappers. However, even in that case Spring’s JDBC support really is quite sufficient. I’ve dealt with custom RDBMS sharding schemes (that didn’t work very well) and would like to work with more distributed data abstractions in the future.

Web front-end

I’ve done several personal projects using Yeoman for scaffolding. I’ve used Sass extensively and despite the relatively recent popularity of using the scripts section in package.json, I still prefer building assets with Gulp. It feels like FRP.

Continuous Testing, Integration and Delivery

I’ve helped a company go from unmaintained unit tests to adopting a full CI cycle using several Dockerized applications (including Jenkins itself) as well as build slaves that provided clean bootstrapped MySQL databases for each Selenium integration test. This required a lot of shell scripting and far reaching extension of the 18 module Maven project.

Automated deployment is hugely interesting to me. If you have work like this for me, get in touch!

Other Skills

Git, Subversion, Eclipse, MySQL, Thymeleaf, JSP’s (in case anyone’s Ctrl+F’ing) and for some reason I’m a Zend certified PHP developer.
